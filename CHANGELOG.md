# Changelog

## 1.0.0 - 2026-06-17

Erster offizieller Release von **OCR Score Rename**.

### Enthalten

- GUI-Anwendung zum Umbenennen gescannter Noten-PDFs anhand OCR
- Instrument-Erkennung mit Synonymlisten (`instruments.yaml`)
- Konsistente Dateibenennung für erkannte und unbekannte Stimmen
- PyInstaller-Builds für macOS und Windows mit gebündeltem Tesseract
- Automatisierte Testsuite mit `pytest`
